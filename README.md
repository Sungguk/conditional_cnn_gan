# Conditional CNN GAN
This project is an exploration of CNN GANs and Conditional CNN GANs on the MNIST dataset.

- CNN: [Convolutional Neural Network](https://en.wikipedia.org/wiki/Convolutional_neural_network).
- GAN: "[Generative Adversarial Networks](https://arxiv.org/abs/1406.2661)" by Ian J. Goodfellow et al.
- Conditional GAN: "[Conditional Generative Adversarial Nets](https://arxiv.org/abs/1411.1784)" by Mehdi Mirza et al.  
- MNIST: [Dataset of handwritten digits](https://en.wikipedia.org/wiki/MNIST_database).
